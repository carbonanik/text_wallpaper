### Prerequisites

- Flutter SDK (^3.8.1)
- Android Studio / Xcode for mobile development
- A tendency to forget things (optional but likely)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/text_wallpaper.git
   cd text_wallpaper
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

### Permissions

BrainPaper needs the following permissions:
- **Storage/Photos**: To save and set wallpapers
- **Wallpaper**: To automatically set your generated wallpaper

Don't worry, we only use these to make your life easier, not to snoop.

## ğŸ“± How to Use

1. **Add Your Sticky Notes**
   - Tap the "Add Sticky" button
   - Type what you need to remember
   - Hit enter or tap "Add"

2. **Customize Your Style**
   - Swipe up the style selector at the bottom
   - Choose your preferred font, color, and layout
   - Make it yours

3. **Set Your Wallpaper**
   - Tap the wallpaper icon in the top right
   - Grant permissions if prompted
   - Boom! Your todos are now your wallpaper

4. **Complete Tasks**
   - Tap tasks to check them off
   - Feel the satisfaction
   - Watch them disappear from your wallpaper

5. **Start Fresh**
   - Tap the clear icon when you're done
   - Confirm you want to clear all tasks
   - Begin again with a clean slate
